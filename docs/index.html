▸ create(expressApp, path_prefix, controller, settings)

For create route with input configuration.

create(expressApp: Object, path_prefix: string, controller: DefaultController, settings: Object)
Version: 0.3.0
Author: Kamontat Chantrachirathumrong
Parameters
expressApp (Object) express app
path_prefix (string) prefix of the path (normally should be /api/vx/ where x is version number)
controller (DefaultController) Default controller or subclass of it
settings (Object = {}) route setting
Name Description
settings.action Array<string> allow action
settings.fixture Object fixture setting
settings.fixture.load boolean auto load fixture in this model?
settings.fixture.clear boolean do we need to clear row in table?
settings.customs Array<Object> array of object for create new path that not default action of DefaultController
settings.customs.name string action name, can be anything.
settings.customs.type string default type, accept get, list, create, update, delete. By method will get path and http-method from those type.
settings.customs.method string define method name in subclass of controller. Must be defined.
settings.customs.path string postfix of path after getting path from type
Example
Route.create(app, "/api", new Controller(), {
     fixture: {
         load: true,
         clear: true
     }
})
